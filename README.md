# XM Solar Library

## Configuración base

### Activar librería XM Solar

```source $(poetry env info -p)/bin/activate```

### Instalación de paquetes

```poetry install```

### Adición de paquete

```poetry add <package>```

### Verificar path poetry env

```poetry env info```

ejemplo: */home/{user}/.cache/pypoetry/virtualenvs/xm-solarlib-C5Mlu42b-py3.10*

### Correr test

```poetry run pytest```
